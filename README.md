# Intergalactic Receiver

The Intergalactic Receiver is an open-source software and hardware project to showcase the work of shader artists. It looks like an old-school radio tuner, but it's connected to a flickering CRT screen.

Mostly the screen shows static, but as you explore the quantum frequency band you encounter "stations" that are abstract procedural animations. They look like something created by alien minds: mathematical, geometric, but somehow also warm and compassionate. They are multi-dimensional streams of friendship and curiosity from outer space.
